SEO改善調査・実装依頼

あなたはSEOとWeb開発の両方に精通したエンジニアです。

対象プロジェクト:

* Repository: https://github.com/135yshr/meow
* Website: https://meow.oreha.dev

背景

現在、「Meow Programming Language」というキーワードでGoogle検索した際に、公式サイトやGitHubリポジトリがほとんど表示されません。

ただし、原因が技術的SEOの問題なのか、コンテンツ不足なのか、Googleの認識不足なのかは未確定です。

推測で修正を行わず、必ず現状を調査した上で改善案を提案してください。

⸻

調査項目

まず以下を調査してください。

サイト調査

* title
* meta description
* canonical
* robots meta
* Open Graph
* Twitter Card
* JSON-LD
* sitemap.xml
* robots.txt
* H1/H2/H3構造
* 内部リンク構造

コンテンツ調査

以下の観点で分析してください。

* Googleが「Meow Programming Language」を固有名詞として認識できる内容になっているか
* 「Meow」だけでなく「Meow Programming Language」という正式名称が十分使われているか
* プログラミング言語サイトとして不足しているページは何か
* 他の言語サイト（Rust、Zig、Gleam、Odinなど）と比較して不足している情報は何か

GitHub調査

READMEを確認し、

* SEO上の問題
* 古いURL
* 不足している情報
* GitHubから公式サイトへの導線

を調査してください。

推奨
